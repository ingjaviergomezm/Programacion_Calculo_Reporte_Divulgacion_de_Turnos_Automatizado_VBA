# Programacion_Calculo_Reporte_Divulgacion_de_Turnos_Automatizado_VBA
Herramienta en Excel + VBA para programación de turnos de 12h, cálculo automático de recargos, análisis comparativo por trabajador (heatmap), KPIs legales y generación segura de reportes PDF y archivos de distribución, alineada con la legislación laboral colombiana.
